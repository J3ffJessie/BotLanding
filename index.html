Get Started

Up and running in minutes

No coding required. Follow these steps and your server will be ready to go.

1

Invite the Bot

Click the Add to Discord button and choose the server you want to add [BOT NAME] to. You'll need the Manage Server permission.

Add to Discord
2

Grant Permissions

Discord will show a permissions screen. [BOT NAME] needs the following permissions to function correctly:

  • Manage Messages — to delete messages during moderation
  • Manage Roles — to assign/remove roles
  • Kick & Ban Members — for moderation commands
  • Send Messages / Embed Links — to respond with rich embeds
  • [Any other required permission] — [reason]
3

Run the Setup Wizard

Once the bot is in your server, type the following command in any channel where the bot has access. This launches an interactive wizard that walks you through configuration.

In any Discord channel 
/setup
4

Configure Your Channels

Tell [BOT NAME] which channels to use for logs, welcome messages, and other features. You can do this through the setup wizard or manually with /config.

Example — set the mod-log channel 
/config modlog #mod-log
Example — set the welcome channel 
/config welcome #welcome
5
